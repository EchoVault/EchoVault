# SMOVE

### Syntax
```
SMOVE source destination member
```

### Module
<span className="acl-category">set</span>

### Categories 
<span className="acl-category">fast</span>
<span className="acl-category">set</span>
<span className="acl-category">write</span>

### Description 
Moves a member from source set to destination set.
