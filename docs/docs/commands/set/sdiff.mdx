# SDIFF

### Syntax
```
SDIFF key [key...]
```

### Module
<span className="acl-category">set</span>

### Categories 
<span className="acl-category">read</span>
<span className="acl-category">set</span>
<span className="acl-category">slow</span>

### Description 
Returns the difference between all the sets in the given keys.
