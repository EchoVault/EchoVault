# ZMPOP

### Syntax
```
ZMPOP key [key ...] <MIN | MAX> [COUNT count]
```

### Module
<span className="acl-category">sortedset</span>

### Categories 
<span className="acl-category">write</span>
<span className="acl-category">slow</span>
<span className="acl-category">sortedset</span>

### Description 
Pop a 'count' elements from multiple sorted sets.
MIN or MAX determines whether to pop elements with the lowest or highest scores respectively.
