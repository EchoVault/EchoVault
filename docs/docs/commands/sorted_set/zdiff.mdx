# ZDIFF

### Syntax
```
ZDIFF key [key...] [WITHSCORES]
```

### Module
<span className="acl-category">sortedset</span>

### Categories 
<span className="acl-category">read</span>
<span className="acl-category">slow</span>
<span className="acl-category">sortedset</span>

### Description 
Computes the difference between all the sorted sets specified in the list of keys and returns the result.

## Options 
- `WITHSCORES` - Whether the returned sorted set should include scores
