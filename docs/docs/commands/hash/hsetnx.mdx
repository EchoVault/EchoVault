# HSETNX

### Syntax
```
HSETNX key field value [field value ...]
```

### Module
<span className="acl-category">hash</span>

### Categories 
<span className="acl-category">fast</span>
<span className="acl-category">hash</span>
<span className="acl-category">write</span>

### Description 
Set hash field value only if the field does not exist.
