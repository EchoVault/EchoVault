# HEXISTS

### Syntax
```
HEXISTS key field
```

### Module
<span className="acl-category">hash</span>

### Categories 
<span className="acl-category">fast</span>
<span className="acl-category">hash</span>
<span className="acl-category">read</span>


### Description 
Returns if field is an existing field in the hash.
