# HDEL

### Syntax
```
HDEL key field [field ...]
```

### Module
<span className="acl-category">hash</span>

### Categories 
<span className="acl-category">fast</span>
<span className="acl-category">hash</span>
<span className="acl-category">write</span>


### Description 
Deletes the specified fields from the hash.
