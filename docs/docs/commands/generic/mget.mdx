# MGET

### Syntax
```
MGET key [key ...]
```

### Module
<span className="acl-category">generic</span>

### Categories
<span className="acl-category">fast</span>
<span className="acl-category">keyspace</span>
<span className="acl-category">read</span>

### Description
Get multiple values from the specified keys.
