# PUBLISH

### Syntax
```
PUBLISH channel message
```

### Module
<span className="acl-category">pubsub</span>

### Categories 
<span className="acl-category">pubsub</span>
<span className="acl-category">fast</span>

### Description 
Publish a message to the specified channel.
